Pig Game
Description

The Pig Game is a simple yet engaging dice game that allows players to take turns rolling dice and accumulating points. The goal is to be the first player to reach a specified score. This project implements JavaScript functionality, allowing for user interaction and game logic.
Features

    Two-player mode
    Simple UI with score display
    Roll dice and take turns
    Option to hold score or end turn
    Responsive design

Technologies Used

    HTML
    CSS
    JavaScript

Getting Started

    Clone the repository:

Code

git clone https://github.com/yourusername/pig-game.git

    Open index.html in a web browser to start the game.

How to Play

    Players take turns rolling a dice.
    The player can choose to either roll again or hold their score.
    If a player rolls a 1, they lose all points for that turn.
    The first player to reach the target score (default is 100 points) wins!
